# Module 2: Rendering Logic 2

- Flow 레이아웃은 요소끼리 최대한 겹치지 않도록 설계되어 있음
- Positioned 레이아웃을 사용하면 이를 손쉽게 구현할 수 있음
- Positioned 레이아웃은 `position` 속성을 통해 사용할 수 있음
  - 기본값은 `static`이며, `relative`, `absolute`, `fixed`, `sticky` 등의 값이 사용 가능함
